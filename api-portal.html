API Developer Portal for Smart Life Companion 2

Welcome to the API Developer Portal for SLC2. Integrate with our enhanced version 2.0 APIs using OAuth 2.0 authentication.

OAuth 2.0 Authentication

All API requests use OAuth 2.0 Authorization Code flow. Follow these steps to set up authentication:

  1. Register your app to get client_id and client_secret.
  2. Redirect to https://auth.smartlifecompanion.com/oauth/authorize?client_id=YOUR_ID&redirect_uri=YOUR_URI&response_type=code&scope=read.
  3. User grants permission and you receive the redirect URI.
  4. POST to https://auth.smartlifecompanion.com/oauth/token with code, client_id, client_secret, and redirect_uri.
  5. Use the access token for API calls.
