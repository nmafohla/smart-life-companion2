OAuth 2.0 Authentication

All API requests use OAuth 2.0 Authorization Code flow.

Steps:

  1. Register your app for client_id and client_secret.
  2. Redirect to https://auth.smartlifecompanion.com/oauth/authorize?client_id=YOUR_ID&redirect_uri=YOUR_URI&response_type=code&scope=read.
  3. User grants permission.
  4. Receive code at redirect_uri.
  5. POST to https://auth.smartlifecompanion.com/oauth/token with code, client_id, client_secret, and redirect_uri.
  6. Use the access token for API calls.
